## What to build

Purchase the Apple Developer Program membership ($99/year). This is the prerequisite for obtaining code signing certificates and notarizing the app for direct distribution. Without this, Gatekeeper will always show the "unverified developer" warning.

Steps:
- Sign in at developer.apple.com with your Apple ID
- Enroll in the Apple Developer Program (individual, not organization)
- Complete identity verification (may take 24–48 hours)
- Confirm enrollment is active and you can access "Certificates, Identifiers & Profiles"

## Acceptance criteria

- [ ] Apple Developer Program enrollment is active
- [ ] Can access Certificates, Identifiers & Profiles in the developer portal
- [ ] Account type is "Individual" (no DUNS number needed)

## Blocked by

None - can start immediately
